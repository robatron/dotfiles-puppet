# dotfiles-puppet

> Automated personal environment setup with Puppet 

## Dependencies

1. [Puppet](http://puppetlabs.com/)
2. [Git](http://git-scm.com/)

## Usage

```bash
git clone git@github.com:robatron/dotfiles-puppet.git ~/.dotfiles
~/.dotfiles/apply.sh
```
